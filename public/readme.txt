This folder is for store files that will be serve via web server and expose to public.

The reserved folder names:
 * Modules - This will be the main folder for any public assets that come with modules. This folder and sub folders will be generate via command line tool (`php rdb`).
 * Modules/[any module name]/* - The module name that have public assets.

Do not create any folder that matches these name, or please leave the *Modules* folder and its sub folder untouch.
